# 🌐 Neon Zerox - Portal

**Neon Zerox** ek modern, fast aur user-friendly online Xerox/Printing portal hai. Is platform ki madad se users apne documents ko online upload karke printing ke liye bhej sakte hain, jisse Xerox shops par lambi lines mein khade hone ka samay bachta hai. Yeh portal photocopy aur document management ko digital aur asaan banata hai.

# <a href="https://birendraverma2019.github.io/Neon-zerox/">Launch Portal</a>

---

## 🚀 Features

- **User-Friendly Interface:** Ekdum clean, modern aur neon-themed responsive UI.
- **Easy Document Upload:** Users apne files (PDFs, Images) asani se upload kar sakte hain.
- **Print Customization:** Pages, copies, color (B&W/Color), aur paper size select karne ka option.
- **Order Tracking:** Apne print orders ka live status check karne ki suvidha.
- **Mobile Responsive:** Har tarah ke devices (Mobile, Tablet, Desktop) par smoothly chalta hai.
